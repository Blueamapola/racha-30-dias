# racha-30-dias
# 🌸 Racha 30 Días

Una app de escritorio kawaii para mantener una racha de 30 días en algo importante para ti.
Hecha en Python + tkinter, con estética rosa y temporizador de 30 minutos por día.

## Características

- ✅ Marca 1 día por jornada real (la racha es de verdad)
- ⏳ Temporizador de 30 min por misión que sobrevive al cierre de la app
- ⏰ Recordatorio diario a la hora que tú elijas
- 🌸 Reloj kawaii análogo en vivo
- 💾 Tus datos se guardan localmente (no se suben a internet)

## Cómo correrlo

Necesitas tener Python 3.10 o superior instalado.

```bash
python racha30.py
```

No requiere instalar dependencias adicionales — todo viene con Python.

## Dónde se guardan tus datos

En `%LOCALAPPDATA%\Racha30\racha30_data.json` (Windows).
Tu progreso no se pierde aunque muevas el archivo `.py`.

## Hecho con

- **Python** 3.10+
- **tkinter** (GUI nativa de Python)
